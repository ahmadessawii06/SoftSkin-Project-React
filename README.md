# 🚀 Getting Started

## 1️⃣ Clone the repository

```
git clone https://github.com/ahmadessawii06/Skin-Project-React.git
```

## 2️⃣ Navigate to project

```
cd Skin-Project-React
```

## 3️⃣ Install dependencies

```
npm install
```

## 4️⃣ Run development server

```
npm run dev
```

---

# 📦 Build Project

To build the project for production:

```
npm run build
```

Preview the build:

```
npm run preview
```

---

# 🚀 Deployment

The project is deployed using **GitHub Pages**.

```
npm run deploy
```
